# Pixel Cat Calendar

## 📥 다운로드 및 설치 안내

### 🍎 Mac (macOS)

1. `.dmg` 파일을 다운로드한 후 실행합니다.
2. 앱 아이콘(고양이)을 **Applications(응용 프로그램)** 폴더로 드래그합니다.
3. 처음 실행 시 **“손상되었기 때문에 열 수 없습니다”**라는 메시지가 나타날 수 있습니다.
   이는 파일이 실제로 손상된 것이 아니라, Apple의 보안 정책(Gatekeeper)으로 인해 차단된 경우입니다.
   아래 단계를 따라 해제해주세요.

#### 🔧 실행 방법 (Gatekeeper 해제)

1. **터미널(Terminal)** 앱을 실행합니다.
2. 아래 명령어를 입력하고 Enter 키를 누릅니다.

   ```
   sudo xattr -rd com.apple.quarantine /Applications/PixelCatCalendar.app
   ```
3. Mac 로그인 비밀번호를 입력하고 Enter 키를 누릅니다.
   (*입력 시 화면에 표시되지 않는 것이 정상입니다.)
4. 아래 명령어를 추가로 입력하고 Enter 키를 누릅니다.

   ```
   sudo codesign --force --deep --sign - --identifier "com.pixelcat.calendar" /Applications/PixelCatCalendar.app
   ```
5. 이제 앱을 정상적으로 실행할 수 있습니다.

---

### 🪟 Windows

1. `.msi` 또는 `.exe` 파일을 다운로드한 후 실행합니다.
2. 실행 중 **“인식할 수 없는 앱”** 경고가 나타나면
   → **추가 정보 → 실행**을 선택하여 진행합니다.

## 📅 캘린더 연동 및 사용 방법

### 🍎 Apple 캘린더

* 앱 설치 후 자동으로 Apple 캘린더와 연동됩니다.
* 일정 시작 **10분 전 알림**이 기본 활성화됩니다.

---

### 🌐 Google 캘린더

※ 원활한 연동을 위해 **사전 사용자 등록이 필요합니다.**
→ 사용하실 **구글 계정을 개발자(ginie)에게 전달해 주세요.**

1. 화면 우측의 **잠자고 있는 고양이 아이콘**을 클릭합니다.
   (*닫혀 있을 경우 한 번 더 클릭하세요.)
2. 하단에 나타나는 **Sign in 버튼**을 클릭합니다.
3. 브라우저가 열리면 Google 계정으로 로그인 후 **계속**을 클릭합니다.
4. 권한 요청 화면에서 **계속**을 눌러 접근 권한을 승인합니다.
5. 이후 캘린더 일정이 앱과 자동 연동되며,
   일정 시작 **10분 전 알림**이 활성화됩니다.
   (*연동 허용 팝업이 나타나면 반드시 승인해주세요.*)

---

💡 **참고**
macOS에서 표시되는 “손상된 파일” 메시지는 실제 파일 문제가 아니라,
공식 인증되지 않은 개발자의 앱을 제한하는 Apple 보안 정책에 따른 것입니다.
