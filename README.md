# 🐱 픽셀 고양이 캘린더 (Pixel Cat Calendar)

내 바탕화면에서 일정을 알려주는 귀여운 픽셀 고양이 캘린더입니다.

## ✨ 주요 기능
- **일정 알람**: 구글/아웃룩/애플 캘린더와 연동하여 일정이 시작되기 10분 전부터 고양이가 깨어나 알려줍니다.
- **귀여운 동작**: 마우스가 움직이면 고양이가 따라오고, 멈추면 앉으며, 밤에는 잠을 잡니다.
- **투명 위젯**: 바탕화면에 녹아드는 투명한 고양이와 깔끔한 캘린더 위젯을 제공합니다.

## 📥 다운로드 및 설치
다른 사용자들은 아래 링크에서 자신의 OS에 맞는 파일을 다운로드할 수 있습니다.
- **Mac**: `.dmg` 파일을 다운로드하여 Applications 폴더로 드래그하세요.
- **Windows**: `.msi` 또는 `.exe` 파일을 다운로드하여 실행하세요.

> **공유 방법**: GitHub Repository의 **Releases** 섹션 링크를 친구들에게 보내주세요!

## 🚀 개발자 공유 가이드
이 프로젝트를 GitHub에 올리면 자동으로 빌드됩니다:
1. GitHub 저장소를 생성하고 이 코드를 Push합니다.
2. `v0.1.0`과 같이 태그를 붙여 Push하면 GitHub Actions가 자동으로 빌드를 시작합니다.
3. 빌드가 완료되면 `Releases` 탭에 설치 파일이 생성됩니다.
