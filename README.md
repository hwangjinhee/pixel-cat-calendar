# 🐱 픽셀 고양이 캘린더 (Pixel Cat Calendar)

내 바탕화면에서 일정을 알려주는 귀여운 픽셀 고양이 캘린더입니다.

## ✨ 주요 기능
- **일정 알람**: 구글/아웃룩/애플 캘린더와 연동하여 일정이 시작되기 10분 전부터 고양이가 깨어나 알려줍니다.
- **귀여운 동작**: 마우스가 움직이면 고양이가 따라오고, 멈추면 앉으며, 밤에는 잠을 잡니다.
- **투명 위젯**: 바탕화면에 녹아드는 투명한 고양이와 깔끔한 캘린더 위젯을 제공합니다.

## 📥 다운로드 및 설치
### 🍎 Mac (macOS)
1. `.dmg` 파일을 다운로드하여 실행한 후, 고양이를 **Applications** 폴더로 드래그합니다.
2. 실행 시 **"손상되었기 때문에 열 수 없다"**는 메시지가 뜨는 경우:
   - 터미널(Terminal) 앱을 엽니다.
   - 아래 명령어를 복사해서 붙여넣고 엔터를 누릅니다.
     ```bash
     sudo xattr -cr /Applications/PixelCatCalendar.app
     ```
   - 만약 위 방법이 안 된다면: **[시스템 설정] > [개인정보 보호 및 보안]** 하단에서 **'확인 없이 열기'**를 클릭하세요.
3. 이제 앱을 정상적으로 열 수 있습니다!

### 🪟 Windows
1. `.msi` 또는 `.exe` 파일을 다운로드하여 실행하세요.
2. "인식할 수 없는 앱" 경고가 뜨면 `추가 정보` -> `실행`을 누르세요.


** macOS에서 "손상된 파일"이라는 메시지가 계속 뜨는 것은 파일 자체가 실제로 깨진 것이 아니라, Apple의 보안
  정책(Gatekeeper)이 "돈을 내고 등록된 개발자가 만든 앱이 아니다"라는 이유로 강제로 차단하고 있기 때문입니다.

  이 문제를 확실하게 뚫고 실행하는 가장 쉬운 방법(마우스만 사용)과 터미널 방법을 다시 정리해 드립니다.

  1. 마우스 우클릭 방법 (가장 추천)
  명령어를 입력할 필요 없이 Apple에서 공식적으로 허용해 주는 우회 방법입니다.

   1. 다운로드한 .dmg를 열고 고양이를 [응용 프로그램(Applications)] 폴더로 옮깁니다.
   2. 응용 프로그램 폴더를 엽니다.
   3. PixelCatCalendar 앱을 찾은 뒤, 그냥 더블 클릭하지 마시고 [마우스 오른쪽 버튼(또는 Control + 클릭)]을 누릅니다.
   4. 메뉴에서 [열기(Open)]를 선택합니다.
   5. 창이 뜨면 아까와 다르게 [열기] 버튼이 보일 것입니다. 그 버튼을 누르면 즉시 실행됩니다.

  ---

  2. 터미널 "최종 보스" 명령어
  만약 위의 방법으로도 안 된다면, macOS가 이 앱에 붙인 모든 보안 꼬리표를 강제로 떼어내는 명령어입니다.

   1. 앱이 반드시 응용 프로그램 폴더에 있어야 합니다.
   2. 터미널을 열고 아래 명령어를 한 줄씩 복사해서 붙여넣으세요.

   1     # 1. 격리 속성 강제 삭제
   2     sudo xattr -rd com.apple.quarantine /Applications/PixelCatCalendar.app
   3     
   4     # 2. 모든 보안 속성 초기화
   5     sudo xattr -cr /Applications/PixelCatCalendar.app
   3. 비밀번호를 입력(화면에는 안 보이지만 입력되고 있음)하고 엔터를 누릅니다.
